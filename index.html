・User thingsとは？

User thingsは誰もが気軽に参加できる全く新しいデリバリーサービスです。

閲覧

User thingsには様々な店舗が登録されています。
あなたのお気に入りのお店を探してみましょう。
見つけた商品はクリックでカートに追加します。

注文

お支払いの際は、ご住所、配達予定時間、商品の税込価格、
配達料金を合わせた注文価格が表示されます。内容に間違いがなければ｢注文する｣をクリックして完了してください。
お支払いは登録されているカードによる自動キャッシュレス決済のため、現金は必要ありません。

配達

User thingsでは他のUserが注文した商品を配達することができます。
たくさんの商品を配達し、多くのUserに評価されると様々な特典をゲットできます。
好きな場所、好きな時間に、あなたもUser thingsで働いてみませんか？
